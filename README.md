# Personal Academic Website - Dickson Akuoko Sarpong

This is a simple static website. It includes:

- `index.html` - main website content
- `styles.css` - design and responsive layout
- `assets/profile.png` - profile image cropped from the provided Google Scholar screenshot

## How to publish with GitHub Pages

1. Create a public GitHub repository named `yourusername.github.io`.
2. Upload `index.html`, `styles.css`, and the `assets` folder.
3. Go to Settings > Pages.
4. Select the main branch and root folder.
5. Your site will appear at `https://yourusername.github.io`.

## Quick edits

- Replace the profile image by putting a new image at `assets/profile.png`.
- Update the Google Scholar, LinkedIn, and email links in `index.html`.
- Add more publications by copying one `<article class="publication">...</article>` block.
